 如何使用

 將 keyBoardHandle.swift 複製到您的專案
 
 然後在 AppDelegate didFinishLaunchingWithOptions
 
 寫上 ChlKayBoardHandle.isEnable = true
 
 搞定
 
 恭喜您，已經完成APP全局的 鍵盤事件處理了
 
 覺得不錯的話，給個星星吧！～
